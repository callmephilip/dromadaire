<h1 align="center">Dromadaire</h1>

1 hump is better than 2

## Local dev

```bash
uv pip install -e .
uv run app
uv run ruff check
```

Use claude in yolo mode with

```
./claude
```

## Running app remotely

```bash
uvx --with git+https://github.com/callmephilip/dromadaire app
```
